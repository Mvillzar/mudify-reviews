Analytics del sitio

📊

Conecta Google Analytics

Para ver datos reales, configura las variables de entorno en Netlify y vincula tu propiedad GA4.

  1. Google Cloud Console → crea un Service Account, descarga el JSON
  2. En Netlify → Site settings → Environment variables, agrega:
    GA4_PROPERTY_ID y GA4_SERVICE_ACCOUNT_JSON
  3. En GA4 → Admin → Property Access Management → agrega el email del service account

Mientras tanto, las métricas de reviews del panel sí están disponibles abajo.
